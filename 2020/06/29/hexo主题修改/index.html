hexo 主题修改

1. 找到自己hexo博客主文件夹

我的是在 D:\blog

进入 system 窗口

2. 主题下载

可以从hexo官网下载： https://hexo.io/themes/

随意演示一个主题的克隆命令

git clone -b master https://github.com/removeif/hexo-theme-amazing.git themes/amazing

themes/ 后的文件名尽量跟主题名一致，方便分别。

找到想要的主题后将命令输入到 system 窗口，然后执行。

下载成功后会在自己的hexo主文件夹里的themes文件夹里有一个amazing文件夹，这就是克隆下来的主题文件。

3. 更换主题

打开自己的hexo主文件夹里的_config.yml文件，找到theme: landscape (landscape 是默认主题)。然后改成 theme: amazing (自己主题的文件夹名字)。

回到 system 窗口按顺序输入并执行以下命令

清除缓存 hexo clean

生成静态文件 hexo g

启动hexo hexo s

最后三句命令可能会出现下面错误

执行完三句命令

4. 简单修改个人信息

找到自己克隆的主题文件夹下的_config.yml文件。然后打开。找到里面的 title: xxxx (头)，author: xxx (作者) 修改即可。当然也有一些要自己修改主页连接的(utl: https://xxx.github.io/ ，改成自己的主页连接即可)

大概就这样。
完工！！！
